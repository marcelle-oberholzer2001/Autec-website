Overview

Efficient size reduction for practical processing duties.

The hammer mill is a flexible processing solution intended for controlled material reduction in mining, mineral processing and workshop environments. It can be configured around the required feed material, target size and operating conditions.

SA Autec Industries can assist with equipment selection, integration and supporting fabrication requirements for a complete installation.

Typical Applications

  • Material preparation and size reduction
  • Process feed conditioning
  • Mining and mineral processing support
